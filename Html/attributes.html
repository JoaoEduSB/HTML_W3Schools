Atributos

Atributos sempre são específicados depois da tag, exemplos:

href - tag < a > define o hyperlink. O atributo href especifica o Url que a página irá seguir.
exemplo: <a href="https://www.w3schools.com">Visite a W3Schools</a>

Resultado = Visite a W3Schools


src - tag < img > vincula uma imagem em uma página HTML

exemplo: <img src="html.jpg" width="500" height="600" alt="Html">

Resultado abaixo:

width (largura), height (altura)

alt é utilizado para posicionar um texto no lugar de uma imagem caso ela não seja exibida por algum erro.


Style ou Estilo

O atributo < style > é utilizado para mudar os estilos de um elemento, como a cor ou o tamanho

Esse é um parágrafo estilizado


< lang > É utilizado entre na tags < html > e < /html > para declarar o idioma da página

Exemplo: < html lang="pt-BR" >


< title > ao passar o mouse encima aparece a palavra

Exemplo: < p title="Sobrepõe" >
