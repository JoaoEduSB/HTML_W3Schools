Oque é o CSS?

Basicamente ele é usado para formatar o layout de um página, podemos controlar cores, tamanhos espaçamentos, posição, e muito mais!

O Css pode ser adicionado das seguintes maneiras:


  • Inline - ou seja em linha
  • Internal - adicionando o < style > na seção da tag < head >
  • External - vinculando os estilos a um arquivo externo através da tag < link >
