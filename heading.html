Os cabeçalhos vão de h1 até h6, sendo h1 o mais importante

e h6 o menos importante, é assim que os navegadores

irão considerar e isso vale para o SEO da página


Curiosidades

Os navegadores adicionam um ligeiro espaço em branco antes e depois de uma tag (< h >)

Não utilize as tags (< h >) apenas com o objetivo de aumentar o tamanho da fonte

PS: podemos aumentar o tamanho de uma tag h usando CSS com o atributo ESTILO ou Style, exemplo abaixo:


< h1 style="font-size:60px;"> Título < /h >


Título - h1 aumentado com o atributo


Título - h1 sem o atributo
