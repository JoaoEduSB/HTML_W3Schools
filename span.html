Span

O span é uma tag Inline usada para marcar uma parte de um texto ou de um documento

Facilmente é estilizada pelo CSS ou manipulada com javascript usando atributo de class ou id

O span é uma tag tipo o elemento < div > mas a div é um elemento block-level enquanto o span é um elemento inline

Qual a diferença entre block-level e inline?

Particularidades de elementos Block-level (div, p, h1 a h6, section, article, header, footer, ul, li, entre outros)

  • Sempre iniciam em uma nova linha.
